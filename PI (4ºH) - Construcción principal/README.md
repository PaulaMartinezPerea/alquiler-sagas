# Plataforma de Alquiler de Sagas

Plataforma web de E-commerce dedicada al alquiler digital de películas con organización por franquicias/sagas. Desarrollada con Next.js, Prisma ORM y MySQL (Docker).

## Prerrequisitos
Para ejecutar este proyecto en local necesitas tener instalado:
- **Node.js** (v18 o superior)
- **Docker Desktop** (para el contenedor de la base de datos)
